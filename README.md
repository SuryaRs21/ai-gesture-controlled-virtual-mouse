AI-Powered Gesture Controlled Virtual Mouse

Technologies: Python, OpenCV, MediaPipe, PyAutoGUI, Computer Vision

Developed a real-time touchless virtual mouse application using Python, OpenCV, MediaPipe, and PyAutoGUI. Implemented hand landmark detection and gesture recognition to control cursor movement, clicking, scrolling, brightness, and volume through intuitive hand gestures, enabling seamless human-computer interaction using only a webcam.
